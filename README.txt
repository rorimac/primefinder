INTRODUCTION
------------

Primefinder is a small program that lists all the prime numbers up to a number specified by the user.


REQUIREMENTS
------------

It requires an installation of Python 2.7 or later on your computer


INSTALLATION
------------

Download the repository or clone it using git from https://www.github.com/rorimac
