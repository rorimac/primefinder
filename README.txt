INTRODUCTION
------------

Primefinder is a small program that lists all the prime numbers up to a number specified by the user.


REQUIREMENTS
------------

Python 2.7 or later
Scipy


INSTALLATION
------------

Download the repository or clone it using git from https://www.github.com/rorimac/primefinder


Known Issues
------------
Error boxes not implemented, errors shown in terminal
Erathosthenes sieve not implemented optimally, smaller primes checked again against larger primes.
